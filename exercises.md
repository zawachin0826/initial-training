    ```python
    def dfdx_approx(x):
        dh = 0.0001
        return (f(x + dh) - f(x)) / dh
    ```

    二次形式に関する微分 $\frac{\partial}{\partial {\bf x}} \left( {\bf x}^{\rm T}{\bf A}{\bf x} \right) = {\bf x}^{\rm T} \left( {\bf A} + {\bf A}^{\rm T} \right)$ を証明してみましょう。
